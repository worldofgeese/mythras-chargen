## Conventions

- Filename: `NNN-kebab-title.md` for original ADRs; `ADR-000N-kebab-title.md` exists for newer MADR-style ADRs. Preserve existing numbering when linking.
- Status lifecycle: `proposed → accepted → implemented → superseded/deprecated`
- Every ADR must cite its source (page number, PDF, plan, or conversation artifact)
