# Adventure Prep Checklists

These notes assume the player characters are using the current strict Mythras sheets.

## Universal Replacements

- Ignore every Strike Rank reference and use Mythras initiative plus Action Points.
- Replace individual weapon skills with the appropriate combat style.
- Use devotional pools as the miracle-use pool.
- Use the relevant rune affinity to launch rune magic and `Devotion` to judge intensity.
- Use only Mythras Core skills at the table.

## GM Skill Substitutions For Adventure Text

| Adventure text term | Use this on the strict sheets |
| --- | --- |
| `Scan`, `Search`, `Listen` | `Perception` |
| `Dodge` | `Evade` |
| `Battle` | `Lore (Strategy and Tactics)` |
| `Bargain`, `Evaluate` | `Commerce` |
| `Cult Lore (X)` | `Lore (X)` |
| `Farm` | `Craft (Farming)` |
| `Herd` | `Craft (Animal Husbandry)` |
| `Homeland Lore (X)` | `Lore (X)` or `Locale` |
| `Intimidate` | `Influence` |
| `Intrigue` | `Deceit` or `Courtesy` |
| `Manage Household` | `Bureaucracy` |
| `Orate` | `Oratory` |
| `Peaceful Cut` | `Craft (Butchery)` |
| `Plant Lore` | `Lore (Plants)` |
| `Animal Lore` | `Lore (Animals)` |
| `Spirit Lore` | `Lore (Spirits)` |
| `Spirit Dance`, `Spirit Travel` | `Trance` |
| `Spirit Combat` | `Binding` for Vishi; otherwise `Willpower` when resisting spirits |
| `Worship (Cult)` | `Devotion (Cult)` on the sheet, rune affinity to cast |
| `Play Instrument (...)` | `Musicianship` |

## A Rough Landing

- Convert likely combatants to Mythras before play:
  - dark trolls
  - scorpion men
  - the krarshtkid
- Use `Perception`, `Influence`, `Willpower`, `Brawn`, and passions for the social pressure scenes in Mernyr's Landing.
- If the text expects a weapon-specific RuneQuest attack roll, use the relevant combat style instead.
- If a spell use is written as a one-use divine spend, deduct from the devotional pool instead.
- If a spirit or supernatural effect appears, use `Binding` only for trained animists; otherwise oppose with `Willpower`, `Devotion`, or the relevant rune affinity.

## A Fire In The Darkness

- Pre-map the fire scene to Mythras skills:
  - `Athletics`
  - `Brawn`
  - `Endurance`
  - `First Aid`
  - `Evade`
  - `Folk Magic`
  - appropriate miracles
- Replace any emergency RuneQuest casting reference with devotional-pool spending.
- Run the investigation through `Perception`, `Insight`, `Influence`, `Deceit`, and `Lore (...)` rather than RuneQuest skill text.
- Convert Lunar combatants as cultists with combat styles and small devotional pools rather than direct RuneQuest blocks.

## The Rainbow Mounds

- Decide which rooms and factions are definitely in play so only those opponents need conversion.
- Collapse trollkin and troll weapon lines into compact combat styles.
- Use Mythras fatigue, armour penalty, hit locations, and devotional pools to keep the dungeon dangerous.
- When the dungeon text expects a legacy skill, use the substitution table above rather than inventing a custom sheet row.
- For improvised knowledge checks about monsters, ruins, or cult traces, prefer `Lore (...)`, `Perception`, `Insight`, and rune-affinity augments.

## Between Adventures

- Refresh devotional pools only when cult access, offerings, and downtime justify it.
- Hand each player the relevant cult one-pager packet.
- Expand the spell worksheet only for spells that actually appeared in play.
- Do not add custom skills to solve adventure edge cases; use the substitution table and note any ambiguous calls in the GM log instead.
