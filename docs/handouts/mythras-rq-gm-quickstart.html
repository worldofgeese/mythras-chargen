Starter Set GM Quickstart

Run RuneQuest Adventures with Mythras Under the Hood

Use the Starter Set for scene intent, NPC motives, lore, and adventure structure. Use Mythras Core for chargen, skill math, initiative, Action Points, combat styles, and Special Effects. Use Adventures in Glorantha plus the cult one-pagers for rune-affinity casting and devotional pools.

Pregens are rebuilt Mythras characters Strict 75-point human rebuild, no patched RuneQuest math.
No Strike Rank Run initiative and Action Points exactly as Mythras does.
Combat styles replace weapon skills Use the style line for attacks, parries, and tactical identity.
No custom skills on sheets Translate legacy adventure calls to Mythras Core skills.
Rune affinity casts miracles Devotional pools replace Rune Points; Devotion still matters.

Read This In Order

This is the minimum reading stack to run the Starter Set in RuneQuest-in-Mythras without paging around blindly at the table.

R1
Read Starter Set Book 4 pages 2-4 first.That gives you Apple Lane, the cast, the campaign frame, and what each adventure is trying to do.
R2
Read the adventure you plan to run straight through.Use Book 4 pages 5-23 for A Rough Landing, 24-47 for A Fire in the Darkness, or 48-75 for The Rainbow Mounds. Read for clues, pacing, factions, and consequences.
R3
Read Mythras Core pages 37-54.That covers the main resolution engine, opposed rolls, standard skills, professional skills, and Passions you will actually call for in play.
R4
Read Mythras Core pages 86-109.You need initiative, Action Points, hit locations, Special Effects, weapons, armor, and movement before the first real fight.
R5
Read Mythras Core pages 121-129 and 179-182, then AiG pages 71-74.That gives you folk magic, divine magic, and the RuneQuest-in-Mythras rule that rune affinity launches miracles while devotional pool pays for them.
R6
Read the converted pregens and their cult handouts together.Use the one-page cult sheets for runes, duties, and miracle tone; use the converted Mythras sheets for all actual rolls and combat handling.
R7
Prep only the opposition the session can actually reach.Convert scene-facing NPCs and monsters now, keep the rest as notes until the players are on that track.

Brief Combat Conversion Example

1. Keep the scene, swap the engine. In the market riot from A Rough Landing, the text gives you drunken trolls, panicked merchants, and a volatile public space. Keep all of that.
2. Replace legacy calls immediately. The scene mentions Scan and Intimidate. At your table those become Perception and usually Influence, with Brawn or a resistance roll if the troll physically shoves someone.
3. If weapons come out, start Mythras combat. Roll initiative, track Action Points, and attack with combat styles instead of weapon percentages or Strike Rank timing.
4. Use Special Effects to make the scene bite. A troll winning the exchange might Trip a hero into smashed stalls; a player winning might Choose Location or Bleed.
5. If a miracle enters play, use AiG casting. Roll the required rune affinity to launch it, spend from devotional pool, and let Devotion guide power and cult tone.
Scene: Starter Set Book 4 pp. 5-8. Combat: Mythras Core pp. 86-100. Rune conversion: Adventures in Glorantha pp. 71-74.
