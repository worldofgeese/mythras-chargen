Player Prep Checklist

Bring this to session zero and review it before each adventure night.

Start Here Combat Path Magic Path Magic in Combat House Rules Source Trail Character Generator

Before session zero

Chargen flow; AiG character creation; source-trail.html

  • Create a character in the Character Generator.
  • Save the JSON export somewhere you can find it.
  • Export or print the PDF character sheet.
  • Write your culture, homeland, career, cult, and magic type in your notes.
  • Read Source Trail so you know the abbreviations.

Before first combat

  • Read Combat Path stages 1-4.
  • Mark your Action Points, hit locations, armour AP, and weapon percentages.
  • Pick two Special Effects you expect to use.
  • Confirm each carried weapon belongs to a combat style.

Before first magical scene

  • Read Magic Path stages 1-2 and your system's stage.
  • For each spell, miracle, or spirit, write the casting roll and resource cost.
  • For each miracle, write source category: common, normal cult, subservient, or associated.
  • For each magical option, write governing rune/skill, MP or pool cost, duration, recovery rule, and source trail.
  • Theists: write Devotional Pool used/available and recovery notes.
  • Animists: write each spirit's ability and slot count.
  • Sorcerers: write quick and shaped profiles for each known spell.

Before each adventure night

  • Open your latest saved JSON/PDF.
  • Review cult obligations, passions, and loyalty before making plans.
  • Use Start Here and Source Trail; consult only books or PDFs you own using the cited page references.
  • Bring one rules question, not ten. Check House Rules first.
