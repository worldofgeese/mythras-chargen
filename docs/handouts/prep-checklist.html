Player Prep Checklist

Use this before session zero, before your first fight, and before game night.

Before session zero

Character creation: Adventures in Glorantha p.23-41

  • Create a character in the Character Generator.
  • Export the PDF character sheet.
  • Save a backup character file from the generator.
  • Write your culture, homeland, career, cult, and magic type in your notes.
  • Open Source Trail and confirm where the book PDFs are on the file server.

Before first combat

  • Read Combat Path stages 1-4.
  • Mark Action Points, hit locations, armour points, and weapon percentages.
  • Pick two Special Effects you expect to use.
  • Confirm each carried weapon belongs to a combat style.

Before first magical scene

  • Read Magic Path stages 1-3 and your own system's stage.
  • For each spell, miracle, or spirit, write the roll and resource cost.
  • Cult Rune Magic characters: write your Devotional Pool used and available, plus recovery notes for each miracle.
  • Animists: write each spirit's ability and bound slot count.
  • Sorcerers: write quick and shaped profiles for each known spell.

Before each adventure night

  • Open your latest character sheet.
  • Review cult obligations, passions, and loyalty before making plans.
  • Bring page numbers for rules questions you already checked: combat actions and special effects, your culture/career skill choices, your cult one-pager, your own spells, spirits, miracles, sorcery shaping, mystic talents, and any GM-approved Monster Island exception you want to use.
  • Check House Rules before asking whether a magic rule differs from the book.
