Read in this order

  1. Build your character in the Character Generator, then save or export the sheet.
  2. Read Combat Path stages 1-4 before the first fight.
  3. Read Magic Path stages 1-3. Stage 3 Folk Magic is mandatory for anyone with cultural or career folk spells; then read the section for your own tradition.
  4. Read House Rules if you use miracles, spirits, sorcery, or cult ranks.
  5. Read Magic in Combat before the first dangerous scene involving magic.
  6. Use Prep Checklist before each session.

If you are mostly a warrior

Start with Combat Path. Know your combat style, weapons, Action Points, armour, and two Special Effects you like.

If you are a cult initiate

Start with Folk Magic, Theism, and House Rules. Bring your selected miracles and rune percentages.

If you are a shaman or spirit worker

Start with Folk Magic and Animism. Bring your bound spirits, slot count, and what each spirit can do.

If you are a sorcerer

Start with Folk Magic and Sorcery. Prepare quick and shaped versions of your three known spells.

Source promise

These handouts are short guides, not a replacement for the books. They mark official Mythras rules, Adventures in Glorantha adaptations, Notes from Pavis cult data, and Hannu house rules. The full abbreviation map is in Source Trail.
